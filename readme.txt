--------------------------------

1.3 Copy C:\Program Files\IBM\ILOG\CPLEX_Studio128\cplex\python\3.6\x64_win64\cplex to C:\Users\Zhicheng\Anaconda3\Lib\site-packages;
1.4 Do a test: import cplex. If no error occurs, that means we're done. If there is any exception, google by yourself and good luck.
